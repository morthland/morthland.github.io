ARES

The Augemented REality Sandtable (ARES) is a prototype sandtable for the military. Traditionally this is done on a large static table with scale models. ARES offers the ability to overlay and project digital scenarios onto a sand surface while being able to manipulate in real time.

I helped develop the tablet application to interface with a future version of this table. Allowing you to create scenes and add real time interactive scenarios to the project. This version also incorporated a hand depth sensor component that allows for gesture control input. I helped develop the custom controls, gestures, table-to-table and interface components.

My portion of this work is not available for public viewing but this video demonstrates some of the talking points to my effort.
