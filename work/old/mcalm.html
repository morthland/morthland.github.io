MCALM

Mobile Collaborative Anxiety Learning and Management system (MCALM) is a phase II Defense Health Program SBIR project. Featuring a comprehensive treatment support tool that can be used along with ongoing therapist-driven treatment to improve tracking, effectiveness, and efficiency of treatment. Using bio-metric hardware and custom algorithms to receive real-time therapy and trigger information.

The system is comprised of a mobile app, bio-metrics wristband, and a online web app. The patient wears a wristband and runs though exercises and daily routines. Therapists can view and respond to bio-metric data and triggers the patient experiences. Providing real time therapy.

I designed the product website along with the entire web app and mobile application. Working with Back-Developers and Project Managers. The only portion I am able to show of this project is the product website front-end. Please see the live website for reference.
