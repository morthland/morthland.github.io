ADAPT-MP

Training games for amputees: An electromyography application.

Delivered to the Defense Health Program in 2015. Auto Diagnostic Adaptive Precision Trainer for Myoelectric Prosthetic's (ADAPT-MP) is a game based approach to training amputees for advanced prosthetic's. Using a bio-metric device to read muscle input and convert it to a training video game. Once Trained on the game a user can progress to an advanced prosthetic. Providing amputees with a fun and intuitive experience before their new limb.

The prototype game is "Sarasaurus" a dinosaur that can jump over land and squat via contracting and releasing your muscles. This project also has a patient tracker and data information mobile app. Giving real-time feedback and information about the user. It also allows a user to request help or contact their doctor directly.
