Program Apollo

Historical website chronicling the adventure of the first manned moon landing, Apollo 11.

I am very fascinated with space and physics so I decided to create this memorial website for the 44th anniversary of the Apollo 11 landing. This was a personal project to try out some new coding techniques and design patterns. While providing insight into the best thing that has ever happened. Walking on the freakin moon!

There were a few things I knew going into this project I wanted: Must be space related, contain custom CSS3 animations, unique scroll navigation, parallax, links to video and resources used. This project was an excellent execution of practice and passion.
